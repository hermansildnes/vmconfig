# vmconfig
Steamlined customization/configuration of shell + prompt + utils.

## Install
```
curl -fsSL https://raw.githubusercontent.com/hermansildnes/vmconfig/main/bootstrap.sh | sh
```
or to change the install directory:
```
curl -fsSL https://raw.githubusercontent.com/hermansildnes/vmconfig/main/bootstrap.sh | sh -s -- --dir /path/to/dir
```
